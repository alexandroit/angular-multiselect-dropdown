# StackBlitz Examples

Standalone StackBlitz projects for the maintained Angular release lines.

Use the single Angular 21 playground for documentation links. The per-example projects remain in `examples/stackblitz/angular-21` for source/reference, but the playground avoids repeated GitHub imports and installs.
