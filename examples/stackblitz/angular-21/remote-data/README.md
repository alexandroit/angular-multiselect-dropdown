# Stackline Angular Multiselect Dropdown - Data from remote API

Angular 21 StackBlitz starter for `@stackline/angular-multiselect-dropdown`.

Open it in StackBlitz:

```text
https://stackblitz.com/fork/github/alexandroit/angular-multiselect-dropdown/tree/master/examples/stackblitz/angular-21/remote-data?startScript=start
```

## What it shows

- `Data from remote API` from the Angular 21 documentation track.
- `<angular-multiselect>` as the primary selector.
- Package version `21.1.14` with Angular `21.2.14`.
- Settings-only skin switching with `settings.skin`.

## Local Run

```bash
npm install
npm start
```
