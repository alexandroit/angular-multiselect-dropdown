# Stackline Angular Multiselect Dropdown - Custom Search from API

Angular 21 StackBlitz starter for `@stackline/angular-multiselect-dropdown`.

Open it in StackBlitz:

```text
https://stackblitz.com/fork/github/alexandroit/angular-multiselect-dropdown/tree/32366299b4e6fd3216be0089b016c18feccf06ed/examples/stackblitz/angular-21/custom-search-api?startScript=start
```

## What it shows

- `Custom Search from API` from the Angular 21 documentation track.
- `<angular-multiselect>` as the primary selector.
- Package version `21.1.14` with Angular `21.2.14`.
- Settings-only skin switching with `settings.skin`.

## Local Run

```bash
npm install
npm start
```
